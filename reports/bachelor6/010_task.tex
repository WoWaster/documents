\section{Постановка задачи}
\label{sec:task}

Целью работы является разработка инфраструктуры для автоматизации проверки задач по курсу \enquote{Теория формальных языков}.
Для её выполнения были поставлены следующие задачи:
\begin{enumerate}
    \item Изучить способы автоматического тестирования учебных задач (раздел~\ref{sec:relatedworks}). % Мб 2.1
    \item Создать базовую инфраструктуру для тестов.
    \item Реализовать решения задач курса.
    \item Обновить используемые инструменты. \note{тут какая-то херь, это housekeeping, это про poetry, dependabot и всё такое}
\end{enumerate}
